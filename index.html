About this project

Goal: Escape a maze using ideas inspired by quantum physics — not a quantum computer simulation, but a teaching metaphor that is easy to explain in a viva. Pick a difficulty mode (Easy through Extreme): each mode has 100 procedural stages. The same mode + stage number always yields the same maze (seed in getLevelDefinition).

What you do: Move on a grid, use Split to exist in several possible cells at once, Measure to collapse to one cell, and use entangled portals (matching colours) to jump across the maze.

Tech: Single-page app with HTML, CSS, and JavaScript only; runs fully in the browser with no backend.

HTML · CSS · JavaScript · No server required

Mission brief

Reach the EXIT using quantum-inspired rules.

  • Move: Arrow keys or WASD
  • Split: Create 2–3 “ghost” positions on adjacent paths (superposition).
  • Measure: Randomly keep one position; the rest disappear (collapse).
  • Portals: Matching symbols — step on one to teleport to its partner (entanglement).
  • Energy: Optional extra score from orbs on the floor.

You win if any of your positions touches the exit — even a ghost.

Game Rules

◎ Objective

Reach the exit (★) on the grid. You complete the stage when any one of your active positions touches the exit — including translucent “ghost” copies from superposition.

⧫ Superposition

Press Split to spread into 2–3 possible locations on adjacent walkable tiles. Each use consumes one split from your budget. Superposition helps you probe forks and cover distance — at the cost of harder control until you measure.

◉ Measurement

When several positions exist, Measure randomly selects one surviving cell and removes the others (wavefunction collapse). Measurement uses a limited charge pool. Portals may still apply immediately after collapse.

⇄ Entanglement

Portals appear as paired tiles with the same Greek symbol. Stepping onto one sends you to its partner elsewhere in the maze — a simple metaphor for linked (entangled) locations. Chained jumps resolve automatically.

✥ Movement rules

Use Arrow keys or WASD. One keypress moves every active token one step in that direction; hitting a wall stops that token only. On touch devices, drag on the maze to steer. Walls are solid. Pause freezes action; Restart reloads the current stage number with a fresh attempt.

⌖ Scoring

Collect optional energy orbs for +50 each. Clearing a level awards a completion bonus that favours fewer moves. Your score is designed to carry across stages in the same run so progression feels rewarding (difficulty and stage layout come from the selected mode).
